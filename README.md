# 簽到程式
若要修改程式,  請註明來源 https://github.com/HansHans135/sign/
---
歡迎加入 https://hbot.fun  </br></br></br>



如何設定?</br>
1.於config.json填入資料</br>
> owner_id填你的id</br>
> prefix填你的前綴</br>
> token就填token</br>
> max每次簽到最高獲得數</br>
> min每次簽到最低獲得數</br>
> new創建帳號初始金額</br>
2.執行`bot.py`</br></br></br>
---
指令大全
---
`help`  __幫助列表__
`new` __創建帳號__
`sign` __簽到__
`me` __看自己有多少錢__
`money <id>` __看別人有多少錢__
`to <id> <錢>` __轉帳給別人__
